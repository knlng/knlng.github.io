Using Mixed Linear Model as a method for Data Analysis¶

24 October 2020

by Linda N.

Overview¶

This is one possible way to use linear mixed models with Python module statsmodels for data analysis. The data used in this notebook is from "http://www-personal.umich.edu/~bwest/rat_pup.dat" which is described on the website "https://www.pythonfordatascience.org/mixed-effects-regression-python/". This data belongs to a study from Jose Pinheiro and Doug Bates where they compared the birth weights of rat pups from different litters. Those litters differ in treatment conditions of mother rats which was either "high dose", "low dose" or "control".

Predict "weight" using statsmodels¶

In [3]:
# enter relevant information into the variables

title_of_file = "weight_of_rat_pups"

dependent_variable = "weight"

# if variable xxx is categorical, it has to be wrapped in "C(xxx)"
# if the reference category aaa should be changed, the categorical variable 
# has to be wrapped in "C(xxx, Treatment(aaa))"
list_of_fixed_effects = ["C(sex, Treatment('Male'))",
                         "litsize",
                         "C(treatment)",
                         "C(treatment):C(sex, Treatment('Male'))"]
random_effect = "litter"
In [4]:
# fixed effects are linked together and separated with a plus
model_independent_var = '+'.join(list_of_fixed_effects)

model = smf.mixedlm(f"{dependent_variable}~1+{model_independent_var}",
                    groups=random_effect, data=df).fit()
print(model.summary())

The result of this example shows either a positive or a negative effect (values in coefficient column) of the respective fixed effect on the weight of rat pups in this study. Furthermore, it shows that the interaction between "treatment" and "sex" ("C(treatment):C(sex, Treatment('Male'))") is not significant as the p-value is not < 0.05.
