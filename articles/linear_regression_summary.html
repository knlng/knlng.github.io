Multiple linear regression¶

22 July 2020

by Linda N.

Overview¶

In contrast to simple linear regression, where there is only one independent variable $x$ and one dependent variable $y$, in multiple linear regression, there are at least two independent variables which results in a two-dimensional array with at least two columns, while $y$ is usually a one-dimensional array.

Linear regression¶

Given are a dependent variable $y$ and three influential variables (independent variables) $x_1 , x_2 , x_3$. The linear regression function is

$$y = a + b_1 x_1 + b_2 x_2 + b_3 x_3$$

or equivalent $$y = \vec{b} \cdot \vec{x}$$ where $$\vec{b} = (a, b_1 , b_2 , b_3 )$$ $$\vec{x} = (1, x_1, x_2, x_3)$$

The objective is to make a prediction and hence to calculate the parameters $a, b_1 , b_2$ and $b_3$.

Example¶

An example of the given values is listed in the following table.

$x_1$ $x_2$ $x_3$ $y$
156 62 24 47
158 42 34 46
160 83 26 46
179 69 51 53
156 74 43 47

The three variables are vectors and can be written in the matrix $X$

$$X = \begin{pmatrix} 1 & 156 & 62 & 24 \\ 1 & 158 & 42 & 34 \\ 1 & 160 & 83 & 26 \\ 1 & 179 & 69 & 51 \\ 1 & 156 & 74 & 43 \end{pmatrix}$$

as well as the vector of the dependent variable $y$

$$\vec{y} = \begin{pmatrix} 47 \\ 46 \\ 46 \\ 53 \\ 47 \end{pmatrix}.$$

With the help of matrix $X$ and vector $\vec{y}$, it is possible to estimate the vector $\vec{b}$ of the parameters:

$$\vec{b} = \begin{pmatrix} a \\ b_1 \\ b_2 \\ b_3 \end{pmatrix} = (X^T * X)^{-1} * X^T * \vec{y}$$

The result will give the regression function as mentioned above:

$$y = a + b_1 x_1 + b_2 x_2 + b_3 x_3$$

Meaning of the parameters¶

  • values $< 0 \rightarrow$ negative effect
  • values $> 0 \rightarrow$ positive effect

If the value of an independent variable is very small, it is possible that this variable does not have an influence on the dependent variable. The p-value allows the estimation whether the independent variable has a significant effect or not.

  • if the p-value is $< 0.05 \rightarrow$ independent variable has a significant effect
  • if the p-value is $> 0.05 \rightarrow$ independent variable does not have significant effect

Using Python package sklearn¶

First, import the required packages numpy and LinearRegression from sklearn.linear_model. Create a list for the matrix $X$ and one for $\vec{y}$. Print the y-intercept and the coefficients.

In [8]:
import numpy as np
from sklearn.linear_model import LinearRegression
X = np.array([
    [156, 62, 24],
    [158, 42, 34],
    [160, 83, 26],
    [179, 69, 51],
    [156, 74, 43]])
y = np.array([47, 46, 46, 53, 47])
reg = LinearRegression().fit(X, y)

print(reg.intercept_)
print(reg.coef_)

6.795673938828877
[0.2405535  0.00222482 0.05438009]

Create a list with the whole result and overwrite it into a column vector.

In [9]:
b_sklearn = np.array([reg.intercept_,
                      reg.coef_[0],
                      reg.coef_[1],
                      reg.coef_[2]])
b_sklearn = b_sklearn.reshape(4, 1)
print(b_sklearn)

[[6.79567394e+00]
 [2.40553503e-01]
 [2.22481730e-03]
 [5.43800938e-02]]

Compare the two methods for equality:¶

In [10]:
# compare the stepwise method with the sklearn method
np.max( np.abs( b-b_sklearn ))
Out[10]:
5.5067062021407764e-12

As the value is very small, both results can be considered to be numerically equal.

Plot the result with matplotlib¶

As it is only possible to plot maximal three-dimensional diagrams, a new example is shown in the following table.

$x_1$ $x_2$ $y$
156 62 47
158 42 46
160 83 46
179 69 53
156 74 47
In [11]:
import numpy as np
from sklearn.linear_model import LinearRegression

# create a list with the values
X = np.array([[156, 62], [158, 42], [160, 83], [179, 69], [156, 74]])
y = np.array([47, 46, 46, 53, 47])

# use LinearRegression from sklearn.linear_model
reg = LinearRegression().fit(X, y)

# calculate interception and coefficients
print(f"interception: {reg.intercept_}")
print(f"coefficients: {reg.coef_}")

interception: 1.5843722264213866
coefficients: [ 0.28582162 -0.00045925]

With this result it is possible to plot the data. Import matplotlib.pyplot and Axes3D from mpl_toolkits.mplot3d.

In [12]:
# import the required packages
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# required to make the 3D plot interactive
%matplotlib notebook

# built the frame
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# scatter plots
ax.scatter(X[:, 0], X[:, 1], y)

# create a linear regression plane with command meshgrid
X_1, X_2 = np.meshgrid(x_1, x_2)
Y = reg.intercept_ + reg.coef_[0] * X_1 + reg.coef_[1] * X_2
ax.plot_wireframe(X_1, X_2, Y, color='red', alpha=0.5)

plt.show()
